Detailed Description

The Scene class is an abstract base class for defining various scenes in the game.

This interface mandates the implementation of core methods essential for scene lifecycle management, including initialization, input handling, updating, rendering, and cleanup. Additionally, it provides methods to check the completion status of the scene and whether the player has won.
